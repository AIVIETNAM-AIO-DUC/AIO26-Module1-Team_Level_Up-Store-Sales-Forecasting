Time-Series Forecasting · Corporación Favorita (Ecuador)

Predicting supermarket sales — without fooling yourself

Real data is rarely "wrong" in a way that throws an error. It's wrong in ways that silently make your model worse and give you no clue why. This guide is split into a Foundations page and four modeling stages — each a focused session that builds on the previous and is scored on the same holdout.

What this guide teaches

Four pillars hold up everything else

  1. No leakage — never use a value you wouldn't have at prediction time.
  2. A gap-free calendar — "7 rows back" must mean "7 days back."
  3. Lags must clear the 16-day horizon — the floor is the horizon length.
  4. A time-respecting holdout — random K-fold cheats; the split must cut in time.

If you internalize those four, the rest is mechanics. The Foundations page defines them in full (plus RMSLE and an interactive calculator); each stage page is one more layer.
